# openpecha-abstract-demo

Demo about OpenPecha abstract text concepts

## Description

Definitions and examples demonstrating OpenPecha's abstract text model:

- **Etext**: A general term for any document read in digital form
- **Work**: A single publication released at a point in time
- **Text**: The main body of a book or piece of writing
- **Volume**: A book forming part of a work or series
- **Poti**: Tibetan term for volume/book (པོ་ཏི་)
- **Collection**: A group of volumes or texts
- **Abstract Text**: A text regarded in terms of its content rather than version
- **Expression**: Any specific edition, publication or version of a text
- **Default Expression**: The digital version selected to represent a text by default
- **Witness**: Any published expression of a text in philology
- **Edition**: A particular form or version of a published text
- **Publication**: Any printed copy of a text
